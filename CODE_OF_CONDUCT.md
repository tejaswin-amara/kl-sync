# 🛡️ Code of Conduct

This project is a community space built by and for KL University students. We want to build something incredibly useful, and we want this project to be welcoming, inclusive, and respectful to everyone who interacts with it.

---

## 🌟 Our Values

- **Respect**: Treat everyone with kindness, regardless of their background, identity, or experience level.
- **Honesty**: Be straightforward and transparent. Disagreements and debates are fine—bad faith arguments are not.
- **Inclusion**: Avoid using heavy insider jargon without explanation. Help each other learn and grow as developers.
- **Safety**: Keep discussions focused on the project. Do not share personal information (real names, phone numbers, addresses, passwords) unless strictly necessary.

## ✅ Expected Behavior

- **Ask good questions**: "How do I set up kl-sync locally?" is welcome. "Why is this broken?" with no context is not.
- **Give thoughtful feedback**: Whether in code reviews or issue discussions, be constructive.
- **Acknowledge mistakes**: "I didn't test this thoroughly" is always better than silence or deflection.
- **Help others**: If you know the answer to a question in an issue or discussion, share it. A rising tide lifts all boats.

## ❌ Unacceptable Behavior

- Harassment, insults, or exclusionary comments based on identity, race, gender, or background.
- Sharing others' private information (email, phone, address, or ERP credentials).
- Spam or repeated off-topic messages.
- Attempts to exploit or sabotage the project.
- Discrimination or gatekeeping (e.g., "You don't know X, so your opinion doesn't matter").

## 🚨 If Something Goes Wrong

- **In public**: Flag it respectfully in the issue or PR. Something like "This comment crosses a line" is often better than immediately escalating.
- **In private**: Email `tejaswinamara@gmail.com` with the details. Be specific about what happened, where it happened, and why it's a problem.
- **Expect action**: Maintainers will investigate all reports and take appropriate steps (which may include a warning, comment deletion, or account suspension from the repository).

## 📍 Scope

This code of conduct applies to:
- All GitHub issues, pull requests, and discussions within the `kl-sync` repository.
- Any official KL Sync communication channels (if a Discord server, Slack, etc., is set up).

It does *not* apply to behavior outside the scope of this project.

---

> **Disagreements are healthy. Rudeness is not.**  
> Let's build something great together.
